פלטפורמת ליווי פיננסי – תכנית עבודה

מערכת מקצה-לקצה: אינטגרציה לדף נחיתה קיים, סליקה, API ממשלתי, דאשבורד לקוח ואבטחת מידע מלאה

תקציר

הפרויקט כולל הקמה מלאה (Backend + Frontend) של מערכת ליווי פיננסי.
עיקרי הפתרון:

  • אינטגרציה לדף נחיתה קיים לקליטת לידים וחיבור למערכת התשלומים.
  • ממשק סליקה (Cardcom) וחיוב חודשי אוטומטי.
  • אינטגרציה ל-API הממשלתי ("הר הביטוח/פנסיה") למשיכת נתוני חסכונות.
  • דאשבורד לקוח להצגת נתונים ותובנות.
  • חתימה דיגיטלית על הסכם ליווי באמצעות Canvas מובנה.
  • אחסון מסמכים ונתונים מאובטח ב-Supabase + Role-Based Access.
  • ניהול מסמכים חתומים ותזכורות אוטומטיות.
